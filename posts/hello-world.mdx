# Hello, World!

Welcome to my first blog post using Next.js and MDX. In this post, we'll explore some of the features of MDX and how it can be used to create rich, interactive blog posts.

## Code Blocks

MDX supports syntax highlighting for code blocks. Here's an example:

```javascript
function greet(name) {
  console.log(`Hello, ${name}!`);
}

greet('World');
```
